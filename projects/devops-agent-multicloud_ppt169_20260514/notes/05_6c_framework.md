简单过一下 6C 框架免得后面频繁提到大家对不上。Context 是上下文和拓扑智能——多源在一次调查里同时关联。Control 是治理和审计——细粒度 IAM 加不可篡改的 audit log。Convenience 是便利性——Agent Space 一处配置全员可用。Collaboration 是协作——告警进来 agent 直接接管，Slack 工单自动协调。Continuous Learning 是持续学习——Skill 三层加学到的 skill。Cost Effective 是成本——按 agent-second 计费，闲时不算钱。下面三个 case 我会分别映射到不同的 C——C1 验证 Collaboration 加 Convenience，C2 验证 Context，C3 验证 Continuous Learning 加 Control。
